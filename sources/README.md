# 参考资料与既有笔记

这里放的是写书用的参考来源，不直接等于正式书稿。

## 1. 官方源码

位置：

- [official/openclaw](official/openclaw)

用途：

- 核验容易过时的事实
- 查看源码链路
- 确认命令、配置、默认行为、插件实现

注意：

- 这个目录本身是独立 Git 仓库
- 需要定期 `git pull`

## 2. 原始笔记

位置：

- [docs-archive-root/README.md](docs-archive-root/README.md)

这部分是你之前学习 OpenClaw 时写下来的原始素材。  
它的价值在于：

- 保留学习轨迹
- 保留原始素材库
- 为后续章节重写和专题抽取提供依据

## 3. 衍生笔记

位置：

- [docs-beginner-pack/README.md](docs-beginner-pack/README.md)
- [docs-curated/README.md](docs-curated/README.md)

这部分不是原始素材，而是从原始笔记整理、压缩、核验出来的衍生层。

可以把它理解成：

- `docs-beginner-pack/`
  - 给小白的压缩阅读版
- `docs-curated/`
  - 给核验、映射、整理用的中间层

## 这层和正式书稿的关系

最简单的理解是：

- `book/`
  - 写给最终读者
- `sources/`
  - 写给写作者和整理者

# 实验验证区

这里不是正式书稿目录，而是书稿制作过程中的实验层。

这一层主要放：

- 主线案例蓝图
- 本机验证手册
- 某一轮真实测试记录
- 还没有沉淀成正式章节的设计稿

- 根目录：
  - 追求阅读顺序稳定
  - 适合系统学习
- `book-lab/`
  - 追求方案可验证
  - 适合边实验边改

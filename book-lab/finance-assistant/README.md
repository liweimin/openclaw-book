# 财务助手实验区

这里放的是“财务运营助手”这条实验线的材料。

这条线不打算先做：

- 投资建议
- 选股建议
- 税务或法律判断

而是先做更适合 OpenClaw、也更容易落地的方向：

- 报销与发票台账
- 收支整理
- 付款 / 回款提醒
- 经营简报

当前建议按这个顺序看：

1. [01-财务运营助手实验蓝图.md](01-财务运营助手实验蓝图.md)
2. [02-报销与发票台账助手验证手册.md](02-报销与发票台账助手验证手册.md)
3. [03-飞书多维表格财务台账-小白保姆级搭建教程.md](03-飞书多维表格财务台账-小白保姆级搭建教程.md)
4. [04-月末关账任务编排助手-可实操场景.md](04-月末关账任务编排助手-可实操场景.md)

## 为什么先从这条线开始

因为这类任务有几个特点：

- 高频
- 真实
- 可验证
- 不依赖复杂网页搜索
- 很适合先用文件、表格、记忆、提醒把流程跑通

## 后续最适合扩展的方向

这条线后面最值得继续长成：

- 回款与付款提醒助手
- 月度经营简报助手
- 预算与费用偏差跟踪助手
- 飞书多维表格财务台账助手

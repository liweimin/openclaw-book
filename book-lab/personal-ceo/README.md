# 个人 CEO Agent 实验区

这里放的是“让默认主 Agent 不只是当助手，而是开始经营你当前工作”的实验材料。

这条线不打算一上来做：

- 宏大人生规划
- 空泛梦想拆解
- 很远期的抽象战略

而是先做更落地的版本：

- 围绕你当前正在做的工作来判断优先级
- 围绕你当前的项目、客户、交付和收入来分配资源
- 围绕“这周怎么推进得更值钱”来做决策

## 这条线和普通助手的区别

普通助手更像：

- 你让我做什么，我就做什么

个人 CEO Agent 更像：

- 先判断什么值得做
- 再判断什么现在最该做
- 再判断什么该停止、该延后、该加码

也就是说，它的核心不是“帮你执行”，而是：

- 帮你经营当前工作
- 帮你看资源分配
- 帮你提高职业和收入质量

## 这条线和 `main-agent-growth` 的关系

- `main-agent-growth`
  - 验证默认主 Agent 能不能先养顺手
- `personal-ceo`
  - 验证默认主 Agent 能不能进一步升级成“经营你当前工作”的个人 CEO

最简单的理解是：

先用 `main-agent-growth` 验证 `main` 是否好用；  
再用这条线验证 `main` 是否值得被提升成你的“工作经营代理”。
